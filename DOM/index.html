Explain DOM

  • The DOM (Document Object Model) in JavaScript is basically a way for JavaScript to interact with a web page.
  • When a browser loads a webpage, it takes the HTML and turns it into a tree-like structure. That structure is called the DOM.
  • Your HTML page → becomes a tree of objects in memory → JavaScript can access and change it with the help of DOM.
  • DOM representation: Document ├── h1 → "Hello" └── p → "Welcome"

Explain DOM Tree

  • The DOM Tree is the way a browser represents an HTML page internally as a hierarchical tree structure.
  • Document(Root) is starting point of the DOM and it represents the entrire page
  • Element nodes are html,body,o,h1 etc
  • Text Nodes are the text inside elements

explain document object

  • The document object in JavaScript is the entry point to the DOM (Document Object Model).
  • It represents the entire web page and gives you access to all HTML elements so you can read, change, or manipulate them. when browser loads an HTML page it creates a global object and it is called document.
  • document = the web page itself as an object

Selecting Elements

  • getElementById (fastest & most common for single element)
  • getElementsByClassName
    • Returns HTMLCollection (like array but not real array)
    • Multiple elements possible
  • getElementsByTagName
    • Selects by tag name (div, p, h1)
    • Returns HTMLCollection
  • querySelector
    • Uses CSS selectors
    • Returns first matching element only
  • querySelectorAll
    • Returns NodeList
    • Selects ALL matching elements
